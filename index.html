Project 5: Comparison of Different Regularization and Variable Selection Techniques

In this project, we will apply and compare the different regularization techniques including Ridge, LASSO, Elastic Net, Smoothly Clipped Absolute Deviation (SCAD), and Square Root Lasso.

In general, an ideal penalty function should result in an estimator with three properties: the estimator is mostly unbiased when the true unknown parameter is large, the resulting estimator is a thresholding rule that sets small estimated coefficients to zero to reduce model complexity, and finally the resulting estimator is continuous in the data to avoid instability in model prediction (Fan and Liu, 2001).

We know that the L2 penalty results in a ridge regression, but it does not have any sparsity in that there are no thresholding rules to set coefficients to zero. Hence, we will see whether or not proposed penalties such as the Elastic Net, SCAD, and Square Root Lasso improve upon the L1 and L2 penalties of Lasso and Ridge.

Creating our own sklearn compliant functions

Since Ridge, Lasso, and Elastic Net are already compliant with the SKlearn GridSearchCV function, we just need to create SKlearn compliant functions for Square Root Lasso and SCAD. Every sklearn compliant estimator needs an __ init __ , fit, get_params and set_params function. The get and set parameter functions can be inherited by subclassing BaseEstimator, so all that is left is to create the __ init __ and fit functions within the two classes.

Square Root Lasso

class SQRTLasso(BaseEstimator, RegressorMixin):
    def __init__(self, alpha=0.01):
        self.alpha = alpha
    
    def fit(self, x, y):
        alpha=self.alpha
        def f_obj(x,y,beta,alpha):
          n =len(x)
          beta = beta.flatten()
          beta = beta.reshape(-1,1)
          output = np.sqrt(1/n*np.sum((y-x.dot(beta))**2)) 
          + alpha*np.sum(np.abs(beta))
          return output

    
        def f_grad(x,y,beta,alpha):
          n=x.shape[0]
          p=x.shape[1]
          beta = beta.flatten()
          beta = beta.reshape(-1,1)
          output = np.array((-1/np.sqrt(n))*np.transpose(x).dot(y-x.dot(beta))/np.sqrt(np.sum((y-x.dot(beta))**2))+alpha*np.sign(beta)).flatten()
          return output
          
        def objective(beta):
          return(f_obj(x,y,beta,alpha))

        def gradient(beta):
          return(f_grad(x,y,beta,alpha))
        
        beta0 = np.ones((x.shape[1],1))
        output = minimize(objective, beta0, method='L-BFGS-B', jac=gradient,options={'gtol': 1e-8, 'maxiter': 50000,'maxls': 25,'disp': True})
        beta = output.x
        self.coef_ = beta

        return self
    def predict(self, x):
        return x.dot(self.coef_)

SCAD

class SCAD(BaseEstimator, RegressorMixin):
    def __init__(self, alpha=1,lam=2):
        self.alpha, self.lam = alpha, lam
  
    def fit(self, x, y):
        alpha = self.alpha
        lam   = self.lam

        @njit
        def scad_penalty(beta_hat, lambda_val, a_val):
            is_linear = (np.abs(beta_hat) <= lambda_val)
            is_quadratic = np.logical_and(lambda_val < np.abs(beta_hat), np.abs(beta_hat) <= a_val * lambda_val)
            is_constant = (a_val * lambda_val) < np.abs(beta_hat)
            
            linear_part = lambda_val * np.abs(beta_hat) * is_linear
            quadratic_part = (2 * a_val * lambda_val * np.abs(beta_hat) - beta_hat**2 - lambda_val**2) / (2 * (a_val - 1)) * is_quadratic
            constant_part = (lambda_val**2 * (a_val + 1)) / 2 * is_constant
            return linear_part + quadratic_part + constant_part

        @njit    
        def scad_derivative(beta_hat, lambda_val, a_val):
            return lambda_val * ((beta_hat <= lambda_val) + (a_val * lambda_val - beta_hat)*((a_val * lambda_val - beta_hat) > 0) / ((a_val - 1) * lambda_val) * (beta_hat > lambda_val))

        @njit
        def scad(beta):
          beta = beta.flatten()
          beta = beta.reshape(-1,1)
          n = len(y)
          return 1/n*np.sum((y-x.dot(beta))**2) + np.sum(scad_penalty(beta,lam,alpha))

        def dscad(beta):
          beta = beta.flatten()
          beta = beta.reshape(-1,1)
          n = len(y)
          return np.array(-2/n*np.transpose(x).dot(y-x.dot(beta))+scad_derivative(beta,lam,alpha)).flatten()
        

        beta0 = np.ones((x.shape[1],1))
        output = minimize(scad, beta0, method='L-BFGS-B', jac=dscad,options={'gtol': 1e-8, 'maxiter': 50000,'maxls': 25,'disp': True})
        beta = output.x
        self.coef_ = beta
        
    def predict(self, x):
        return x.dot(self.coef_)

Soft Thresholding

We can apply a technique called soft thresholding to turn certain values that meet a threshold to zero that otherwise would be considered nonzero values, but unlike hard thresholding, the coefficients whose absolute values are greater than the threshold will have their value reduced by the threshold. Unfortunately, soft thresholding is the penalized least squares estimate with the L1 penalty in the general least squares and likelihood settings, and it only works on functions with sharp corners, which does not include Ridge, but we can apply it to the rest of the penalties, which all have a sharp corner in their penalty function.

Note the soft vs hard vs SCAD penalty functions

For a derivation of the threshold itself, if we suppose that, in order to estimate the noise free coefficient www, the problem can be formulated as y=w+ny = w + ny=w+n, where yyy is the noisy coefficient and nnn is the noise. A maximum a posterior (MAP) estimator is used, denoted w^(y)\hat w(y)w^(y), that is based on the pdf of www.
Moreover, y=w+ny = w + ny=w+n is normally distributed with a mean of www, and to obtain the soft thresholding function, most academic sources use the formula: w^(y)=sign(y)⋅(∣y∣−λ)+\hat w(y) = \text{sign}(y)\cdot (|y|- \lambda)_+w^(y)=sign(y)⋅(∣y∣−λ)+​

Implementation in Python

To use soft-thresholding we can define a function in python to perform thresholding on the estimated beta coefficients.

# this is equivalent to having 3 conditional if else statements
def soft_threshold(data, value):
  new_coef = []
  for i in range(len(data)):
	# avoid dividing by 0 errors
    if (data[i] == 0):
      new_coef.append(0)
    else:
      # perform the thresholding in 1 line
      new_coef.append(data[i]/np.abs(data[i]) * np.maximum(np.abs(data[i]) - value, 0))
  return np.array(new_coef)

For the value of the threshold, we will simply use the penalty term that controls the L1 penalty, which will be α\alphaα for Lasso and Square Root Lasso. For SCAD and Elastic Net the penalization term is not just α\alphaα, so for Elastic Net we will use α⋅l1\alpha\cdot l1α⋅l1_ratio and for SCAD we will use α⋅λ\alpha\cdot \lambdaα⋅λ.

Otsu’s Method for Thresholding

In addition to Soft Thresholding, one could also use Otsu’s method to further reduce the number of coefficients to zero. While it is primarily used in image segmentation, the same method can be applied to variable selection by dividing the observations based on percentiles and choosing to set the values in some lower percentile thresholds to zero.

However, the sparsity patterns obtained from some of the regularization methods ended up being very close to the true sparsity of β∗\beta^*β∗, so I did not implement this method for this project.

Square Root Lasso
